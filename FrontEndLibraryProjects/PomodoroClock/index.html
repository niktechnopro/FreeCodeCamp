Focus!

Based on Pomodoro Technique

The Pomodoro Technique is a method that can "help anyone to focus."

Instructions:

  • Choose a task to be accomplished.
  • Set the time for Work and Break
  • Work on the task until work session is over (work timer countdown)
  • Take a short break (rest timer countdown)
  • After every four work sessions take a long break
  • Enjoy funny "facts" about Chuck Norris
